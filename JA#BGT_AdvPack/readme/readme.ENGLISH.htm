1 Introduction

This mod expands upon the content of the BG1 part of a BGT game.
There is a lot of content in BG1 that did not make it into the final game. The mod "BG1 Unfinished Business" restores some of them, but there are a lot more still. In addition, there are many instances in the game where the impression arises that something is missing or that it was not implemented completely.

This Adventure Pack adds to such points in the game. A lot of missing content not included in "BG1 UB" is restored by this mod, and situations that seemed meager are expanded upon. Still, there would be a lot more unused content to be found in the game. The more I worked with the BG1 files, the more instances I encountered.

Additionally to restorations and expansions this mod introduces some of my own ideas. For a detailed list of the content please refer to the section "Components" below.

Cheers; Jarl


2 Installation & Compatibility

Unpack the content of the zip-folder into your BGT main game folder and start the Setup-JA#BGT_AdvPack.exe bei double-clicking. You have to start a new game after the installation!

From Version 0.7 on, compatibility with BG1NPC Project (concerning the Xzar-Niemain content please note info below), BG1 Romantic Encounters, BGQE, Garrick Tales of a Troubadour and Xvart Caverns will be provided for the following installation order:

  • bgqe
  • bg1npc
  • bg1re
  • garrick-tt
  • bg1ub (content installed by bg1ub will be skipped automatically by JAP)
  • JAP
  • Xvart Caverns.


Compatibility to NTotSC v2.x (or higher) is also provided. If NTotSC is installed before JAP, the overlapping restored content will be skipped by JAP. If the content of JAP is wanted, the according components of NTotSC should not be installed.

Following content will be skipped if already installed by NTotSC or UB:
  • Keelor & Nasha
  • Sylast, the Fallen Paladin
  • Pilar and Gheldehar
  • Llyndellyn the Archer
  • Nim Furlwing and her Hounds
  • Mal-Kalen, the Ghost from Ulcaster
  • Elfsong Tavern Movie


Due to the compatibility problems with BG1NPC content concerning the Niemain additions, the additional content concerning Niemain and the Zentarim is now (Version 0.7 or higher) an own component: "Second meeting and dialogue improvements for Niemain and the Zhentarim".
Upon installation of this component, the player decides to see the content of JAP in the game which will replace the BG1NPC content.

Compatibility with Baldurdash's "Special Item" store of Aldeth Sashenstar is provided: the store can also be accessed in chapter 8.

Compatibility with "Jarl's BGT Tweak Pack" is provided.

REQUIREMENTS
  • Main component of Infinity Animations needs to be installed.
  • BG1 Unfinished Business #19 "Minor Dialogue Restorations" is required.
  • BG1 Unfinished Business #21 "Store, Tavern and Inn Fixes and Restorations" is required.
  • The function "CharmPatch" of TobEx is recommended.
  • The "Jondalar Fix for BGT" is not needed, because JAP offers a similar fix itself.

4. Hints and Notes

Note the following to see as much mod content as possible:

  • Spare / defend Aldeth's life in the Cloud Peaks to see all the extended restored material for the Mearchants' League.
  • In chapter 8 return to Jhasso inside the Seven Suns.
  • Talk to Husam whenever you might see him.
  • For pacifying Caldo and Krumm it needs a bit of imagination. ;)
  • The plot inside the Ducal Palace is different now and more extended!

Spoiler:

  • If fighting Marl inside Feldepost's Inn the guards appear. How can I prevent this?
    >Keep to the unwritten laws for tavern brawls, meaning: no weapons and only one opponent against one! Best to turn off group-AI during this fight.
  • The Iron Throne Building is locked. How do I get inside?
    >Ask the drunk Husam for advice. Also, search inside the houses nearby.
  • How can the desk inside the Mearchants' League be opened?
    >You need to pick-pocket the key from Zorl. Zorl can be sent away after talking to Irlentree on the first level, then the desk can be opened without consequences.
  • Sarevok's Journal no longer is with Cythandria in the Iron Throne Building?
    >No, Sarevok's journal will be inside the Ducal Palace.
  • Why is the scene inside the Ducal Palace not starting?
    >To start the scene, you need to go and pick Sarevok's journal from the second level. Nila will help you with this. Also, if expecting help from Husam, you need to grant him and his Shadow Thieves access to the palace, first (via the cellars).
  • How can I spare Caldo and Krumm?
    >Try the spell "charm person".
